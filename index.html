Some programming languages I share for you

Scratch is a visual programming language that makes it easy to create interactive stories, games, and animations 😊 It's a great way to introduce children to programming and develop their creativity and problem-solving skills 🤔 🥳

Python

Python is a powerful yet easy-to-learn language for beginners 🥰 It is used for a wide variety of tasks, including web development, data science, machine learning, scientific computing, and system administration 😄

C++ is a powerful, compiled language that can be used to create efficient software 🥳 It is a good first language for developers because it teaches the fundamentals of programming, such as data types and control flow 🧐

GML

GML is a proprietary scripting language for GameMaker 💽 It is dynamically typed and similar to JavaScript and C-like languages 🔍 GML can be used to create 2D and 3D games for Windows, macOS, Linux, HTML5, Android, and iOS 📲 🥳

HTML

HTML is the standard markup language for creating web pages 🖥️ It uses tags to describe the structure of a web page, such as headings, paragraphs, lists, and images 🧐 HTML is often used with CSS 🖌️ and JavaScript 💽 to create visually appealing and interactive web pages 🥳
